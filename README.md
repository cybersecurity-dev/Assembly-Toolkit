# Assembly Toolkit

<details>
 
 <summary>For Debian, Ubuntu, Linux Mint</summary>
  
 ```bash
 sudo apt-get update && sudo apt-get install -y nasm
 ```
 </details>


<details>
 
 <summary>For RHEL, Fedora, AlmaLinux</summary>
  
 ```bash
 sudo dnf install -y nasm 
 ```
 </details>

 <details>
 <summary>For Arch, Manjaro, EndeavourOS</summary>
  
 ```bash
sudo pacman -S nasm
 ```
 </details>
